# Code of conduct

UllrAI communities should be welcoming, respectful, and useful.

## Expected behavior

- Discuss ideas and code constructively.
- Respect different backgrounds, experience levels, and viewpoints.
- Focus criticism on the work, not the person.
- Accept feedback and correct mistakes in good faith.

Harassment, discrimination, threats, personal attacks, and deliberate
disruption are not acceptable.

Maintainers may edit or remove contributions and restrict participation when
necessary to protect the community. Report conduct concerns privately to
[hello@ullrai.com](mailto:hello@ullrai.com).
