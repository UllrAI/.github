# Security policy

## Reporting a vulnerability

Please do not disclose security vulnerabilities through public GitHub issues.

Email [hello@ullrai.com](mailto:hello@ullrai.com) with:

- the affected repository, product, and version or commit;
- a clear description of the vulnerability and its impact;
- reproduction steps or a proof of concept, when available;
- any suggested remediation.

We will acknowledge the report and coordinate disclosure after we have assessed
the issue. Please avoid accessing, changing, or retaining data that does not
belong to you while investigating.
